# morsepractice

Simple tool to aid your practicing morse code sending.

## Build

### install dependencies
```
npm install
```

### build desktop app
```
npm run tauri build
```

### build webapp
```
npm run build
```
webapp version is hosted on [https://bivoje.github.io/morsepractice/]

## Usage

Use 'j' or 'k' in keyboard for input.

You can drag & drop text files to load them.
